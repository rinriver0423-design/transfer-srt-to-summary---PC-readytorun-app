# SRT Summarizer

SRT字幕双语摘要生成工具 - 基于 Electron + MiniMax API 的桌面应用。

## 功能
- 导入 SRT 字幕文件
- 自动识别中英文字幕
- 调用 MiniMax API 生成双语摘要

## 使用方法
1. 填写 MiniMax API Key 和 Group ID
2. 导入 SRT 文件
3. 点击"生成双语摘要"

## 技术栈
- Electron 33
- MiniMax Text-01 API
- electron-builder (portable exe)

## 注意事项
- 可执行文件位于 `dist/SRT Summarizer 1.0.0.exe`
- 源码位于 `srt-summarizer -main/` 目录
